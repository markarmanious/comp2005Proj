        1. Users can register an account with a username and password

          Implication of the "Account Registration" use case.

          Access the sign up page from the login screen.

          Attempt to create an account with an existing username and receive and error message.

        5. User passwords have a minimum length of six characters

          Passwords cannot be any shorter than six characters to ensure account security. Attempting to create an account with a password below the minimum length will result in an error.

      2. User Login
        1. Users can authenticate with Login/Password

          User logins and passwords have to be supported, and there may be special users who are guests or not formally registered for the course. Associated with feature 2: User password authentication.

          If a user enters incorrect login credentials several times, they will be unable to attempt another login for a set period of time. Multiple consecutive lockouts will result in an increased lockout duration.
